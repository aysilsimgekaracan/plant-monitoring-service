# plant-monitoring-service

## Installation

#### Requirements
- Python3 (Tested on Python 3.12)
- FastAPI ```pip3 install "fastapi[standard]"```

### Create Virtual Environment And Install Requirements

```python3 -m venv venv```
```source ./venv/bin/activate```
```pip3 install -r requirements.txt```


## Run

```fastapi dev main.py```
